# 轨道跃迁 · Orbit Leap

一键操作的霓虹风太空街机游戏。Phaser 3 + Vite + TypeScript，全部贴图与音效程序化生成，零外部资源。

支持作为 **PWA** 安装到桌面 / 手机主屏，安装后**完全离线可玩**；竖屏 540×960 设计，FIT 等比缩放适配手机、平板与桌面宽屏（宽屏下带页面级星空背景与霓虹画框）。连应用图标都是程序化生成的（`scripts/gen-icons.mjs`，纯 Node 手写 PNG 编码）。

## 玩法

- 你绕着行星的轨道环公转，**点击 / 空格** 脱离轨道，沿切线飞出（虚线即弹道方向）。
- 飞行中受重力下坠，碰到下一颗行星的轨道环即被捕获，继续公转。
- 不断向上攀升；撞上行星本体、触雷、被超新星吞没、或被下方上涌的虚空追上，游戏结束。

### 行星与机制

| 元素 | 说明 |
| --- | --- |
| 普通行星 | 气流带 / 陨石坑 / 漩涡三种程序化表面 |
| 脉冲行星 | 轨道环半径呼吸伸缩 |
| 金色行星 | 弹射速度 1.5 倍，金色拖尾 |
| 不稳定行星 | 捕获即点燃引信（环上有倒计时弧），2.3 秒后爆炸 |
| 轨道地雷 | 红色尖刺，在轨道环上公转，碰到即死；200m 以上出现，越高越多 |
| 星星 | +100 分 |
| 护盾（六边形） | 抵挡一次致命撞击（地雷 / 本体 / 爆炸；虚空除外），碎裂后短暂无敌 |

### 计分

高度（米）+ 星星 ×100 + 行星捕获 ×50 ×连击倍率。7 秒内连续跃迁到新行星可叠连击。
每 400m 进入新的高度区域（近地轨道 → 电离层 → 深空 → 星云带 → 幽蓝远域 → 虚空边境），星云换色、虚空加速。
最佳纪录存 localStorage。

## 运行

```bash
npm install
npm run dev      # 开发（不注册 Service Worker，避免缓存干扰热更新）
npm run build    # 构建到 dist/，自动生成 sw.js（预缓存全部产物）
npm run preview  # 本地预览生产构建（PWA / 离线能力需在此或正式部署下验证）
```

### PWA 说明

- `vite.config.ts` 内置零依赖插件：构建后扫描 `dist/` 生成 `sw.js`，缓存名取内容哈希，新版本上线自动接管并清理旧缓存。
- 安装即预缓存全部资源（游戏运行时不发任何额外请求），断网后照常游玩。
- 图标若需重新生成：`node scripts/gen-icons.mjs`（输出到 `public/icons/`）。

## 结构

```
src/
  main.ts              # 游戏入口与配置（540×960 竖屏，FIT 缩放）
  constants.ts         # 尺寸、霓虹配色、高度区域
  sfx.ts               # WebAudio 程序化音效（振荡器 + 噪声缓冲）
  scenes/
    BootScene.ts       # 程序化贴图：行星表面、星云、暗角、地雷、护盾、冲击波环等
    MenuScene.ts       # 标题菜单 + 公转演示 + 机制图例
    GameScene.ts       # 核心玩法：轨道力学、四种行星、地雷、护盾、连击、区域、虚空
    GameOverScene.ts   # 结算与重开
```
